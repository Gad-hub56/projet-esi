Programme du Cours - Développement Web

Ce cours couvre les fondamentaux du développement web, du HTML et CSS à la gestion des bases de données et aux frameworks modernes.

Contenu du cours

  • Introduction au HTML et CSS
  • JavaScript et interactivité
  • Conception responsive et UI/UX
  • Gestion des bases de données web (MySQL, MongoDB)
  • Frameworks front-end et back-end (React, Node.js)
  • Projet pratique : création d’un site web dynamique

Enseignants

  • M. Ouedraogo - Développement Front-End
  • Mme Sanou - Gestion des bases de données
